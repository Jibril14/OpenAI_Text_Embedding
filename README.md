# Embedding Wikepedia articles for Semantic search

This project demonstrate how text documents are cleaned, processs and turned into vector representation
for indexing and semantic search.


### How to Set up the application
Open terminal 

  2. cd OpenAI_Text_Embedding
  3. python3 -m venv venv
  4. venv/bin/activate
  5. pip install -r requirements.txt
  6. python main.py
